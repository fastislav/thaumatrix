# 🔮 Thaumatrix

**Unified Esoteric Resonance Engine**

Где нумерология, Матрица Судьбы, Таро, руны и биоритмы сходятся в один голос.

## ✨ Что это

Thaumatrix — это не просто набор калькуляторов. Это система, которая:
- Запускает параллельно 5+ эзотерических расчётов
- Ищет пересечения между системами
- Генерирует **резонансный итог** — текст, который читается как откровение

## 🧩 Модули

1. **Нумерология** — Число Жизненного Пути, Судьбы, Души
2. **Матрица Судьбы** — 22 аркана, кармические линии
3. **Таро** — Аркан рождения, Личный год
4. **Руны** — Руна рождения и года
5. **Биоритмы** — 4 цикла (физический, эмоциональный, интеллектуальный, интуитивный)

## 🚀 Как использовать

1. Откройте [thaumatrix.com](https://thaumatrix.com) (или thaumatrix.ru)
2. Введите имя и дату рождения
3. Нажмите «Активировать резонанс»
4. Получите результаты всех систем + итоговое послание
